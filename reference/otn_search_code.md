# Search OTN database by project code.

Search OTN database by project code.

## Usage

``` r
otn_search_code(code, server = "otn")
```

## Arguments

- code:

  Character. Project code for which to search. Supports partial
  matching.

- server:

  Character. URL of the OTN-style Plone server. Defaults to the main OTN
  server at <https://members.oceantrack.org>.
