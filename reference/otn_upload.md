# Upload a file to the Plone CMS

Upload a file to the Plone CMS

## Usage

``` r
otn_upload(file_path, project)
```

## Arguments

- file_path:

  Character. Location of the file to be uploaded.

- project:

  Character. The project code.

## Value

Silently returns a vector of file locations.
