# Internal function to list small OTN data bases

Provides access to the data that is on the main OTN members page. Login
is not required.

## Usage

``` r
.otn_list(server = NULL, type)
```

## Arguments

- server:

  Character. URL of the OTN-style Plone server. Defaults to the main OTN
  server at <https://members.oceantrack.org>.

- type:

  Character. Internal data base to retrieve.
