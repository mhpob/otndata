# Search OTN database by institution.

Search OTN database by institution.

## Usage

``` r
otn_search_institution(institution, server = "otn")
```

## Arguments

- institution:

  Character. Institution for which to search. DOES NOT support partial
  matching.

- server:

  Character. URL of the OTN-style Plone server. Defaults to the main OTN
  server at <https://members.oceantrack.org>.
