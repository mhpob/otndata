# Search OTN database by project country.

Search OTN database by project country.

## Usage

``` r
otn_search_country(country, server = "otn")
```

## Arguments

- country:

  Character. Project country for which to search. DOES NOT support
  partial matching.

- server:

  Character. URL of the OTN-style Plone server. Defaults to the main OTN
  server at <https://members.oceantrack.org>.
