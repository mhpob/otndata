# Search OTN database by project contact.

Search OTN database by project contact.

## Usage

``` r
otn_search_contact(contact, server = "otn")
```

## Arguments

- contact:

  Character. Project contact for which to search. DOES NOT support
  partial matching.

- server:

  Character. URL of the OTN-style Plone server. Defaults to the main OTN
  server at <https://members.oceantrack.org>.
