# Search OTN database by local node.

Search OTN database by local node.

## Usage

``` r
otn_search_node(node, server = "otn")
```

## Arguments

- node:

  Character. OTN node for which to search. DOES NOT support partial
  matching.

- server:

  Character. URL of the OTN-style Plone server. Defaults to the main OTN
  server at <https://members.oceantrack.org>.
