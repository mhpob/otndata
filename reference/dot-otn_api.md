# Call OTN API endpoints.

Call OTN API endpoints.

## Usage

``` r
.otn_api(endpoint, server = otn_global$server_url)
```

## Arguments

- endpoint:

  Character. API endpoint to call.

- server:

  URL of the network's Plone CMS
